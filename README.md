# 🧠 Pikman AI — Суперагент на облаке

**Pikman AI** — это мощный AI-агент для Android, который работает на облачных дедиках, имеет полный контроль над телефоном, встроенный браузер, консоль для кода и самоулучшается ежедневно.

## 🎯 Ключевые особенности

✅ **Облачные вычисления** — 13B-70B LLM на бесплатных дедиках  
✅ **Полное управление телефоном** — ADB, скриншоты, навигация  
✅ **Встроенный браузер** — DuckDuckGo, Google, скачивание видео  
✅ **Консоль и песочница** — выполнение кода в изолированной среде  
✅ **Голосовой ввод/вывод** — TTS с мужским голосом  
✅ **Автоматическая регистрация** — на 20+ бесплатных сервисах  
✅ **Память и обучение** — запоминает пользователя, дообучается  
✅ **Самоулучшение** — ежедневное совершенствование  
✅ **Бело-оранжевый дизайн** — минимализм + анимации (20%)  
✅ **Бэкап в Telegram** — сохранение в чат "Избранное"

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- Docker Desktop
- Android Studio
- Git

### Установка

```bash
# 1. Клонируй репозиторий
git clone https://github.com/kkjj1488i-prog/Pikman-agent.git
cd Pikman-agent

# 2. Копируй .env
cp .env.example .env

# 3. Запусти бэкенд
docker-compose -f docker-compose.prod.yml up -d

# 4. Собери APK
cd frontend
npm install
expo build:android

# 5. Установи на телефон
# Скачай APK из артефактов и установи
```

## 🔄 Самоулучшение (Self-Improvement)

Каждый день в 3:00 UTC агент:

1. Анализирует ошибки и запросы
2. Предлагает улучшения (отдельное окно)
3. Тестирует в виртуальной среде
4. Коммитит в GitHub (закрытый репо)
5. Деплоит новую версию
6. Отправляет отчёт в чат
